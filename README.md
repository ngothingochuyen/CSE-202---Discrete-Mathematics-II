COURSE DESCRIPTION
This course helps students gain an in-depth understanding of graph concepts and algorithms. Students will learn about the basic concepts, properties, and representations of graphs, as well as their applications in scientific and practical fields. The course also includes several algorithms for solving or partially solving graph theory problems. Students will develop skills in modeling, analyzing, and solving problems using graph and tree structures and algorithms.

COURSE OBJECTIVES
At the end of this course, students will be able to:

Use graph theory concepts and theorems in the field of Information Technology.
Apply algorithms to solve problems related to graph theory.
Analyze and model real-world problems related to graph theory in the field of Information Technology.
Select appropriate data structures to represent graphs of problems in Information Technology using a programming language.
Apply graph algorithms and related data structures to specific problems using common programming languages.
